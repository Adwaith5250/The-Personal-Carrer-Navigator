## üß¨ The Analysis Algorithm

The core logic follows a strict heuristic pipeline:

1.  **Decomposition:** The target role is broken down into atomic skill units.
2.  **Mapping:** Your profile is mapped against these units using semantic similarity (e.g., "React.js" matches with "Frontend Library").
3.  **Scoring:** Gaps are weighted based on market frequency and importance.
4.  **Synthesis:** A natural language summary is generated to explain *why* a gap exists and *how* to fix it.
